LIRIBOT is a siri version for Node.js implemented to serve to 
seek data from APIs namely

- SPOTIFY
- OMDB
- BANDS IN TOWN

HOW IT WORKS
-------------

use the following key words to search data in the terminal after the command
node liribot.js 

- spotify-this-song (song name)
-----------------------------------------------------------------------------
eg: spotify-this-song its my life
